## Troubleshooting

### Buttons Not Showing
**Solution**: Refresh the page (Ctrl+R or Cmd+R)

### Hear Question Not Working
**Check**:
1. Speaker volume is on
2. Browser audio is not muted
3. System volume is on
4. Try different browser

### Record Answer Not Working
**Check**:
1. Microphone is connected
2. Browser has microphone permission
3. Microphone is not muted
4. Try Chrome or Firefox (best support)

### Microphone Permission
**Chrome**:
- Settings → Privacy → Microphone
- Allow for localhost:3000

**Firefox**:
- Preferences → Privacy → Microphone
- Allow for localhost:3000

**Safari**:
- System Preferences → Security & Privacy
- Microphone → Allow

**Edge**:
- Settings → Privacy → Microphone
- Allow for localhost:3000

## Tips for Best Results

### Using Record Answer
1. **Speak clearly** - Enunciate each word
2. **Normal pace** - Don't speak too fast or slow
3. **Complete sentences** - Better transcription
4. **Pause between thoughts** - Helps with punctuation
5. **Quiet environment** - Reduce background noise
6. **Good microphone** - Better quality = better results
7. **Review text** - Check before submitting
8. **Edit if needed** - You can correct manually

### Using Hear Question
1. **Listen carefully** - Understand the full question
2. **Take notes** - Write down key points
3. **Ask for clarification** - Re-read the question
4. **Use as study aid** - Helps understand better
